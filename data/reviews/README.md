# Reviews Data

This folder contains CSV files with reviews from Google and Booking.com.

## Files

- `google-reviews.csv` - Reviews from Google
- `booking-reviews.csv` - Reviews from Booking.com

## CSV Format

Each CSV file should have the following columns:
- `name` - Reviewer name
- `rating` - Rating (1-5 for Google, 1-10 for Booking.com)
- `date` - Review date
- `comment` - Review text
- `country` - Reviewer country (optional)

## Usage

These CSV files are used to dynamically display reviews on the website.
